# MNIST Digit Classification (ANN & CNN)

## 📌 Overview
This project classifies handwritten digits using deep learning models.

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras

## ⚙️ Approach
- Preprocessed image data (normalization, reshaping)
- Built ANN model (~94% accuracy)
- Built CNN model (~99% accuracy)
- Tuned epochs and dropout

## 📊 Results
- ANN: ~94%
- CNN: ~99%

## 🧠 Key Learning
CNN performs better than ANN for image data due to spatial feature extraction.
