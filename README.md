# Handwritten Digit Recognition using ANN & CNN

## 📌 Overview
This project focuses on classifying handwritten digits using Deep Learning models on the MNIST dataset.

Initially, an Artificial Neural Network (ANN) model was implemented and later improved using a Convolutional Neural Network (CNN) architecture.

The project demonstrates the complete deep learning workflow including preprocessing, model building, training, evaluation, hyperparameter tuning, and prediction.

## ⚙️ Approach
- Preprocessed image data (normalization, reshaping)
- Built ANN model (~94% accuracy)
- Built CNN model (~99% accuracy)
- Tuned epochs and dropout

## 📊 Results
- ANN: ~94%
- CNN: ~99%
- CNN performed significantly better than ANN for image classification tasks.

## 🧠 Key Learning
- Importance of preprocessing image data
- Hyperparameter tuning improves performance
- CNN is highly effective for image classification
- Understanding overfitting and dropout
